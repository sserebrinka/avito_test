# High Priority

1. Фильтр по городу: выбран Санкт-Петербург, но отображаются вакансии из других городов (Москва, Самара)  
Обоснование выбора приоритета:
Нарушение логики фильтров, пользователь получает нерелевантные результаты

2. Фильтр “Можно удаленно”, но показываются и вакансии только офисного формата  
Обоснование выбора приоритета:
Нарушение логики фильтров, пользователь получает не подходящие результаты

3. Выбрано направление “Поддержка пользователей”, но показываются вакансии Data Science и другие позиции  
Обоснование выбора приоритета:
Выдача не соответствует запросу, пользователю трудно найти подходящую роль

4. Число вакансий по направлению “Клиентский сервис” указано как 2, но фактически отображается 1  
Обоснование выбора приоритета:
Неверная статистика, ошибки подсчета

5. Сообщение “Ничего не нашлось” появляется посреди подборки, хотя результаты ниже есть  
Обоснование выбора приоритета:
Нарушение UX, кажется что поиск завершен

6. Поле “Команда” показывает, что вакансий нет, при этом результаты на странице есть  
Обоснование выбора приоритета:
Противоречие данных интерфейса, скорее всего ошибка бэкенда

7. Выбран параметр “руководящая роль”, но отображаются позиции, не являющиеся руководящими  
Обоснование выбора приоритета:
Несоответствие фильтра реальному результату
